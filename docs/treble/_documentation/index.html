Introduction

Treble is One Page theme. We have created flexibile HTML5 layout which will allow you to edit and adjust Treble to your needs.

Treble includes Ajax loaded pages, for example Portfolio items.

We have created Blog section if you want to extend your website. With latest blog which is implemented in one page and Blog list separate layout with blog posts.

Treble is responsive

We adapted look and functionalities on all devices and views. You can expect that navigation will look as native mobile application when looking your website using mobile devices, or tablets.

Cross browser

Yes, treble will look the same in Firefox, Chrome, Safari, Internet explorer, Android browser. Here is the list:

  • Firefox
  • Chrome
  • Internet explorer 9+
  • Safari
  • Adroid browser
  • Mobile Firefox beta

Clean and organized

You will find that everything has it's place in Treble. Name convention same for id's, classes, and file structure. This will help you to find your way and adapt.

You can use Treble just by using HTML, we have prepared large amount of elements to be used, no need to write CSS.

But if you need to adapt look, you can use CSS or if you prefere LESS.

Requirements

You will need only basic stuff. Web server and domain. Treble is running only on HTML5, CSS3, JS.

There is only one part of Treble which require PHP. Twitter. So if you need Twitter posts to be implemented, you will need to have PHP on server.
